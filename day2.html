День 2. Оформление сайта

Сайт написан без вордпрессов, тильд, редимагов и макета, поэтому я столкнулся с некоторыми проблемами:

  • Только с третьего раза написал нормальную адаптивную сетку для главной страницы, а остальные делал на ее основе

  • Не предусмотрел вертикальные блоки: навигация | контент | футер. Намучался потом оборачивать все в дополнительные теги

  • Сайт разрастался стихийно и количество CSS свойств тоже. Для следующего месяца обязательно это учту.

Список не закончен и будет пополняться.

Сетка для больших экранов

Разделёна на три блока: Боковые для навигации и белого пространства, центральный – для контента. Использовал flex boxes. В центральном блоке две колонки для описания и картинок/параграфов, разделял с помощью grid.

Сетка мобилок

При разрешении экрана меньше 800px все аккуратненько выстраевается в ряд.
